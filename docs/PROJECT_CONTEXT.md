# QuantEdge — 项目背景文档

## 1. 项目目标

构建一个综合量化投资平台，集成以下四大模块：

1. **量化评分** — 对标的进行多因子打分
2. **组合回测** — 历史策略回测与组合绩效分析
3. **实时监控** — 价格、指标、信号的实时追踪与告警
4. **投资日志** — 交易记录、决策复盘、持仓管理

## 2. 目标市场

- **美股**
- **港股**

## 3. 数据粒度

- **日线起步**
- 架构需预留 **分钟级** 扩展能力（存储模型、时间索引、接口抽象均需兼容）

## 5. 关键设计决策

- **ETF 与个股的指标体系完全分离**
  - 个股使用基本面 + 技术面混合因子
  - ETF 使用与个股不同的指标集（例如跟踪误差、溢价折价、底层持仓穿透等），
    不与个股共用同一套评分逻辑
- 该决策影响：数据模型、因子库、评分引擎、前端展示组件都需按资产类型分流
